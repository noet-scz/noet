# noet

Цензуроустойчивое сообщество. Личность: криптографический ключ, контент: в IPFS, имена: в собственном реестре noet.

## Расширение

Позволяет открывать имена noet прямо в адресной строке браузера, без DNS и без серверов.

**Установка (Chromium/Vivaldi/Brave/Edge):**

1. Открой `chrome://extensions` → включи **Режим разработчика**
2. Нажми **Загрузить распакованное** → выбери папку `extension/`
3. Иконка появится в панели браузера — клик включает/выключает

**Или скачай готовый `.crx`** из папки `dist/` и перетащи на страницу расширений.

После установки имена noet открываются как обычные сайты.
